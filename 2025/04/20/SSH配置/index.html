创建SSH

本地创建ssh配置文件

第一步

桌面右键打开git bash

检查是否存在SSH

看是否存在 id_rsa 和 id_rsa.pub文件，如果存在，说明已经有SSH Key 可以直接跳过看第三步

第二步

创建SSH

如果没有SSH key使用一下指令

写完后直接回车就行

第三步

获取SSH key公钥

然后复制输出结果也就是 SSH key

创建SSH密钥

进入 github 官网 点击右上角头像 找到 setting 打开到个人设置页面

找到 SSH and GPG keys 进入到页面 点击绿色按钮 New SSH key

添加SSH密钥

谁便起一个名字方便记忆 然后在key粘贴你的SSH key 然后点击Add SSH key
